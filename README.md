# Django-blog

## 项目名称:
Django开发个人博客

## 项目描述：
基于Django框架实现的水果商城项目

## 开发工具准备:
操作系统windows10<br>
Python版本:3.6.5<br>
开发工具:pycharm<br>
第三方模块:django2.1.7<br>
数据库:sqlite3<br>
